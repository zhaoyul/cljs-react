* VS Code 安装指南
对于 VS Code, 你只需要安装 [[https://calva.io/][Calva]] 就可以开始了:
1. 打开 VS Code
2. 按 ~Ctrl/Cmd + Shift + P~
3. 搜索 "安装扩展"
4. 在扩展市场搜索 "Calva"
5. 点击 ~安装~ 按钮
6. 完成!

#+CAPTION: 在 VS Code 上安装 Calva
[[file:./public/assets/calva.png]]
